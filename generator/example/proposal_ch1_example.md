บทนำ

## 1.1 ที่มาและความสำคัญ {#1.1-ที่มาและความสำคัญ}

ในปัจจุบัน มหาวิทยาลัยอุบลราชธานีกำหนดให้นักศึกษาที่กู้ยืมกองทุนเงินให้กู้ยืมเพื่อการศึกษาและนักศึกษาทุนต้องเข้าร่วมกิจกรรมเพื่อสะสมหน่วยกิตกิจกรรมให้ครบตามเงื่อนไขที่กำหนดไว้ อย่างไรก็ตาม การประกาศรับสมัครนักศึกษาเข้าร่วมกิจกรรมส่วนใหญ่อยู่ในรูปแบบการเผยแพร่ผ่านเพจ Facebook ของหน่วยงานต่าง ๆ ภายในมหาวิทยาลัย ซึ่งอาจส่งผลให้นักศึกษาที่ไม่ได้ติดตามข่าวสารผ่านช่องทางดังกล่าวพลาดโอกาสในการสมัครเข้าร่วมกิจกรรมเพื่อสะสมหน่วยกิตกิจกรรม

นอกจากนี้ กระบวนการรับสมัครและการบันทึกหน่วยกิตกิจกรรมของเจ้าหน้าที่ผู้ปฏิบัติงานด้านกิจกรรมนักศึกษายังคงใช้การจัดการเอกสารด้วยตนเอง (Manual) ซึ่งก่อให้เกิดปัญหาหลายประการ เช่น ความล่าช้าในการดำเนินงาน ความไม่ตรงกันของรายชื่อผู้เข้าร่วมกิจกรรมระหว่างเอกสารและใบลงชื่อ การสูญหายของเอกสารเก่า รวมถึงความยากลำบากในการตรวจสอบประวัติการยื่นคำร้องขอบันทึกหน่วยกิตกิจกรรมย้อนหลัง ส่งผลให้การบริหารจัดการกิจกรรมนักศึกษาไม่เป็นไปอย่างมีประสิทธิภาพ

จากปัญหาดังกล่าว ผู้พัฒนาจึงตระหนักถึงความสำคัญของการนำเทคโนโลยีมาประยุกต์ใช้เพื่อแก้ไขข้อจำกัดที่เกิดขึ้น จึงได้พัฒนาเว็บแอปพลิเคชันระบบรับสมัครนักศึกษาเข้าร่วมกิจกรรมในรูปแบบออนไลน์ เพื่อเป็นศูนย์กลางในการเผยแพร่ข่าวสารการรับสมัครกิจกรรม การจัดการข้อมูลการยื่นคำร้องขอบันทึกหน่วยกิตกิจกรรม และการจัดเก็บเอกสารอย่างเป็นระบบ เว็บแอปพลิเคชันดังกล่าวจะช่วยเพิ่มความสะดวกให้กับนักศึกษาในการติดตามข่าวสารและสมัครเข้าร่วมกิจกรรมได้อย่างทั่วถึง รวมทั้งช่วยลดข้อผิดพลาดและเพิ่มความรวดเร็วในการดำเนินงานของเจ้าหน้าที่ผู้รับผิดชอบ

## 1.2 วัตถุประสงค์ {#1.2-วัตถุประสงค์}

1.2.1   เพื่อพัฒนาเว็บแอปพลิเคชันระบบรับสมัครนักศึกษาเข้าร่วมกิจกรรมในรูปแบบออนไลน์ ที่รองรับผู้ใช้งาน 4 กลุ่มหลัก ได้แก่ นักศึกษา ผู้รับผิดชอบโครงการ เจ้าหน้าที่คณะ และผู้ดูแลระบบ
1.2.2   เพื่ออำนวยความสะดวกให้แก่นักศึกษาในการติดตามข่าวสารและสมัครเข้าร่วมกิจกรรมเพื่อสะสมหน่วยกิตกิจกรรมได้อย่างทั่วถึง
1.2.3   เพื่อเพิ่มประสิทธิภาพในการจัดการข้อมูลการรับสมัครและการยื่นคำร้องขอบันทึกหน่วยกิตกิจกรรม พร้อมลดความล่าช้าและข้อผิดพลาดจากการจัดการเอกสารในรูปแบบเดิมของเจ้าหน้าที่

## 1.3 ขอบเขตของโครงงาน {#1.3-ขอบเขตของโครงงาน}

### 1.3.1 นักศึกษา

1. สมัครสมาชิกได้
2. เข้าสู่ระบบได้
3. แก้ไขข้อมูลส่วนตัวได้
4. เรียกดูรายการและรายละเอียดประกาศกิจกรรมได้
5. สมัคร/ยกเลิกการสมัครเข้าร่วมกิจกรรมได้
6. เรียกดูรายชื่อผู้สมัครเข้าร่วมกิจกรรมได้
7. เรียกดูประวัติการเข้าร่วมกิจกรรมได้
8. เรียกดูสรุปข้อมูลหน่วยกิตกิจกรรมตามปีการศึกษาได้

## 1.4 ประโยชน์ที่คาดว่าจะได้รับ {#1.4-ประโยชน์ที่คาดว่าจะได้รับ}

1.4.1   นักศึกษาสามารถติดตามข่าวสารและสมัครเข้าร่วมกิจกรรมเพื่อสะสมหน่วยกิตกิจกรรมได้อย่างสะดวกและรวดเร็ว
1.4.2   เจ้าหน้าที่สามารถจัดการข้อมูลการรับสมัคร การยื่นคำร้อง และตรวจสอบประวัติหน่วยกิตกิจกรรมได้อย่างเป็นระบบ
1.4.3   ลดความล่าช้า ความผิดพลาด และการสูญหายของเอกสาร พร้อมเพิ่มประสิทธิภาพในการดำเนินงานและการสรุปข้อมูลกิจกรรมผ่านระบบแดชบอร์ด

### 1.5.1 ฮาร์ดแวร์ (Hardware)

1. เครื่องคอมพิวเตอร์ส่วนบุคคล

### 1.5.2 ซอฟต์แวร์ (Software)

1. Visual Studio Code ใช้เป็นเครื่องมือสำหรับการพัฒนาเว็บแอปพลิเคชัน
2. HTML ใช้ในการสร้างโครงสร้างและเนื้อหาของหน้าเว็บ

5. Django Framework เป็นเฟรมเวิร์กโอเพนซอร์สสำหรับพัฒนาเว็บแอปพลิเคชันด้วยภาษา Python
